### Requisitos para utilização do sistema
  - Node
  - Backend do sistema (api)
  ``` git clone https://github.com/pabloguedesc/node_store_backend.git ```

### 1. Instalar as depedências.
  ```npm install```
	ou
  ```yarn```

### 2. Rodar script do sistema:
  ````npm start````
	ou
  ```yarn start```
